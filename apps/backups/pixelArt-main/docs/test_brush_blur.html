Brush Blur Test

This test verifies that when the brush tool is selected, the blur value is set to at least 1.

Test 1: Select brush tool with blur = 0

Initial state: brushBlur = 0

Test 2: Select brush tool with blur = 1

Initial state: brushBlur = 1

Test 3: Select brush tool with blur = 5

Initial state: brushBlur = 5

Test 4: Select pencil tool (should not affect blur)

Initial state: brushBlur = 0
